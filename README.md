## Brief Description of the System

The Online Voting System is a web-based application developed to manage and conduct digital elections efficiently. 
The system allows administrators to manage voters, candidates, and positions, while enabling the recording and counting of votes in a centralized database. 
The admin dashboard provides a quick overview of the election statistics, including the total number of voters, candidates, positions, and votes.

## Steps to Run the Project

1. Install a local server environment such as XAMPP for my case.
2. Download or copy the project folder (kisubi-website).
3. Place the project folder inside the htdocs directory in XAMPP.
4. Open the XAMPP Control Panel.
5. Start the Apache and MySQL services.
6. Open a web browser and navigate to:
   http://localhost/kisubi-website

## Database Import Instructions

1. Open your web browser and go to:
   http://localhost/phpmyadmin
2. Click New to create a new database.
3. Enter a database name (for example: `kisubi_voting`).
4. Click Create.
5. Select the newly created database.
6. Click the Import tab.
7. Click Choose File and select the `.sql` file included in the project.
8. Click Go to import the database.
9. After the import completes, the system will be ready to use.
